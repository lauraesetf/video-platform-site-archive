# video-platform-site-archive
该项目用于整理影视平台及相关网站的公开基础信息，作为长期资料归档与查询参考使用。仓库内容仅限站点层面的客观资料记录，不涉及影视资源存储、播放或分发行为。所有信息来源于公开网络环境，整理目的在于信息汇总、结构化管理与历史留存。本项目不对站点内容合法性、用户体验或运营情况作出评价，仅作为中性的信息整理项目存在。
